Les pronoms

1) les pronoms allocutifs :

Ce sont les pronoms qui représentent les personnes participant au discours. Le shindzuani en connaît deux types, résumés dans le tableau suivant:

1ère pers. sg. 2e pers. sg. 1ère pers. pl. 2e pers. pl. autonomes wami "moi" wawe "toi" wasi "nous" * wanuy "vous" non-autonomes sujet object ni- "je"
tsi- "je" (acc.)
tsi- "je (conditionnel négatif) -ni- "me, moi" u- "tu" -hu- "te, toi" ri- "nous" ri- "nous" mu- "vous" mu-...ni "vous"

* Les jeunes (jusqu'à une vingtaine d'années) disent watru

Dans le classement du lexique, on ne trouvera que les formes autonomes; les autres ont été considérées comme des variantes de ces formes.

3) les présentatifs :

Comme pour les démonstratifs, les présenta tifs connaissent trois degrés: proche, intermédiatre et élotgné. Le premier peut être traduit par votct, le ou la ou les votci, Le deuxième et le troisième par voilà, le ou la voilà, les voilà. Voici les principaux accords:
