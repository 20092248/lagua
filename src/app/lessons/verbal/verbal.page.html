Le système verbal Présentation

En shindzuani, un verbe, dans sa forme conjuguée, est accompagné de nombreux affixes (préfixes et suffixes). Ce sont: les pré-préfixes verbaux (ppv), les préfIXes verbaux (PY), les relateurs (rel.), les aspecto-temporels (tps), les auxiliatifs (aux.), les infixes-objets 10), les extensions ou dérivatifs (dér.), les suffixes verbaux (sfv) et les post-suffixes verbaux (psfv). Ces affixes se situent de part et d'autre du radical verbal (RAD) et se présentent de la manière suivante :

ppv-PV-rel.-tps-aux.-IO- RAD-dér.-sfv-psfv

1) Les pré-préfixes verbaux peuvent être soit la marque de la négation (tsi- : nég. + je, ka- : pour les autres personnes ou classes), soit la marque de l'injonction (na-).

2) Les préfixes verbaux sont des marques de personnes ou de classes obligatoirement présentes. On les considère, souvent, comme les pronoms-sujets des verbes.

3) Le relateur est une marque de relatif (-a-).

4) Les aspecto-temporels sont, comme leur nom l'indique, les marques de l'aspect et du temps. Ce sont: -ø- pour l'accompli affirmatif ou le présent relatif ou encore l'accompli relatif affIrmatif, -a- pour l'accompli négatif, -si- ou -su- pour l'inaccompli-présent, et -tso- pour le futur.

5) Les auxiliatifs sont des éléments à base de radicaux verbaux qui assument une fonction d'auxiliation. Ils donnent une indication sur le déroulement de l'action indiquée par le verbe: ils ont une valeur essentiellement aspectuelle. On distingue:

a) -dungu- : cet auxiliatif évoque une idée de répétition ou de harcèlement. Ex. : lisha, usidungunitaâbisha! : "arrête, ne m'embête pas sans arrêt !" (= arrête de m'embêter sans arrêt) [-dunga "suivre, harceler"]

b) -ahi- : cet auxiliatif indique que l'action indiquée par le verbe doit précéder une autre. Ex. : mwahija, mutsoona : "si vous venez, vous verrez"

c) -jo- : cet auxiliatif évoque l'idée de venir faire l'action indiquée par le verbe. Ex. : nisijófanya ihâzi : "je viens faire le travail". [-ja "venir"]

d) -ako- : cet auxiliatif indique que l'action indiquée par le verbe a connu une certaine durée ou bien qu'elle était simultanée à une autre action ou qu'elle était en train de se réaliser. Ex. : anipára nakotséha "il m'a trouvé en train de rire" : nakópindra : "je m'habillais" [-ka "être"]

e) -lo- : cet auxiliatif indique qu'on a été faire l'action indiquée par le verbe. Il ne s'emploie qu'à l'accompli. Ex. : alomútolea ihadisi "il a été lui raconter l'histoire" [-lawa "sortir, partir"]

f) mo- : cet auxiliatif évoque l'idée de simultanée, mais de façon durative, répétitive. On pourrait le traduire par "chaque fois que...". Il suit presque toujours l'auxiliatif -ahi-. Ex. : nahímofanya hazi, tsivéndze múntru anitaâbísha : "quand je travaille, je n'aime pas qu'on m'embête"

g) -andro- : cet auxiliatif indique le risque encouru en accomplissant une action donnée. Ex. : fahámu wandrókodza ! : "fais attention, tu risques de te blesser !"

h) -ondro- : cet auxiliatif évoque l'idée d'aller faire l'action indiquée par le verbe. Ex. : nisondromúrema : "je vais le frapper". Il peut se combiner avec l'auxiliatif -ako-. Ex. : nakondromúrema : "j'allais le frapper". [-endra "aller, marcher"]

i) -aparo- : cet auxiliatif évoque l'idée d'avoir déjà réalisé au moins une fois l'action indiquée par le verbe, ou de ne l'avoir jamais réalisée. Ex. : nisífikiri amba naparomúona : "je crois que je l'ai déjà vu", muntru úwo tsapáromukia nyangu nádzwala : "je n'ai jamais entendu parler de cette personne depuis que je suis né". L'auxiliatif peut se combiner avec -jo- et -ondro-. Ex. : naparojomúona :"j'ai déjà eu l'occasion de venir lui rendre visite", tsapárondromuona : "je n'ai jamais été lui rendre visite". [-para "trouver, obtenir"]

j) -shindro- : cet auxiliatif traduit une possibilité d'accomplir l'action indiquée par le verbe. Il s'emploie exclusivement avec l'habituel (voir ci-dessous). Ex. : wami ushindromuóna : "je peux le voir, j'ai la possibilité de le voir"... [-shindra "pouvoir"]

k) -atsungu- : cet auxiliatif permet de déclencher immédiatement une autre action. Ex. : natsúngumuona, tsibáma : "dès que je l'ai vu, je me suis évanoui".

6) Les inftxes-objets sont des pronoms substitutifs en accord de classe avec le nom auquel ils se réfèrent.

7) Les extensions sont des éléments dérivationnels. Elles ont une valeur aspectuelle ou de voix (voix active, voix moyenne, voix passive). On distingue:

a) l'applicatif (Ap.) -i-, -e-. Cette extension permet d'introduire deux compléments d'objet. Il présente aussi une valeur bénéfactive, instrumentale ou locative. Ex. : -fanya "faire", -fanyia "faire pour le compte de". Anifanyia ihazi : "il m'a fait le travail", -dza "accoucher", -dzalia "accoucher à". Adzalia lopitáli : elle a accouché à l'hôpital"...

b) le moyen-neutre (Mn.) -ih-, -eh-, -h-. Cette extension présente deux valeurs:

- le sujet de la phrase est en même temps l'objet de l'action indiquée par le verbe. Ex. : -vundza "casser", -vundziha "se casser". - le moyen-neutre peut indiquer aussi que le procès est faisable. Ex. : -la "manger", -liha "être mangeable"

c) le causatif (Cs.) -is-, -es-, -dz-, -z-. Cette extension a pour rôle de faire faire l'action indiquée par le verbe, par une tierce personne. Ex. : -la "manger", -lisa "faire manger", -lia "pleurer", -lidza "faire pleurer", -lewa "être ivre", -leza "enivrer"

d) le passif (Ps.) -w-, -iw-, -ew-, -lw-. Ex. : -funga "attacher", -fungwa "être attaché", -lisa "faire manger", -lisiwa "être nourri", -bua "ouvrir", -bulwa "être ouvert"

e) l'associatif ou le réciproque (Rp.) -an-. Cette extension indique que l'action du verbe est assumée par plusieurs participants. Ex. : -vendza "aimer", -vendzana "s'aimer (réciproquement)"

8) Les suffixes verbaux sont des voyelles qui indiquent l'aspect, le mode (indicatif, subjonctif). Pour les verbes disyllabiques, ces voyelles sont, à l'accompli affirmatif et à l'habituel négatif (voir ci-dessous), souvent en harmonie avec la voyelle du radical. Pour les cas des verbes où ce phénomène se présente, il sera précisé dans le lexique, en mettant notamment entre parenthèses, la forme de l'accompli en classe 1.

9) Les post-suffixes verbaux sont: soit des relatifs, soit des suffixes-objets, si la place de l'inftxe-objet est déjà occupée.
